# HangoutX

HangoutX is a modern social hangout platform inspired by Spotify's clean and immersive user interface. The platform is designed to connect people through communities, messaging, gaming, and interactive social experiences.

## Features

* Modern dark-themed Spotify-inspired design
* User profiles and customization
* Real-time messaging interface
* Arcade gaming hub
* Community discovery and engagement
* XP and rewards system
* Responsive layout for desktop and mobile
* Local storage data persistence

## Tech Stack

* HTML5
* CSS3
* JavaScript
* LocalStorage

## Vision

The goal of HangoutX is to create a digital space where users can socialize, play games, join communities, and build meaningful connections in a fun and engaging environment.
